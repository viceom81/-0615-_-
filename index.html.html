구글 스프레드시트 완전 연동 가이드

본 웹앱에서 수집한 설문 결과를 선생님의 구글 드라이브 스프레드시트에 직접 누적하여 기록하는 방법입니다.

✔ 연동 설정 정보 입력

아래 단계에 따라 구글 앱스 스크립트(Google Apps Script)를 배포한 뒤 웹앱 주소(Web App URL)를 입력해 주세요.

현재: 로컬 브라우저 저장 모드로 동작 중입니다 (스프레드시트 전송 대기 상태)

간단한 4단계 연동 과정

1

구글 스프레드시트 생성 및 구조화

구글 드라이브에서 새 스프레드시트를 만들고 첫 행(헤더)에 각각 다음 컬럼을 입력합니다.

A열: [타임스탬프] | B열: [성함] | C열: [소속 구분] | D열: [담당/직무] | E열: [참석 여부] | F열: [의견/건의사항]
2

앱스 스크립트(Apps Script) 실행 및 코드 붙여넣기

스프레드시트 상단 메뉴에서 [확장 프로그램] > [Apps Script]를 클릭합니다.

기존 코드를 모두 지우고, 아래의 소스코드를 그대로 복사하여 붙여넣으세요.

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // 현재 날짜 및 시간 구하기
    var timestamp = new Date();
    
    // 데이터 행 추가
    sheet.appendRow([
      timestamp, 
      data.name, 
      data.dept, 
      data.role, 
      data.attendance, 
      data.comment
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({
      "result": "success",
      "message": "데이터가 성공적으로 추가되었습니다."
    })).setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({
      "result": "error",
      "message": error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}
3

스크립트 배포 및 웹앱 주소 생성

  1. 앱스 스크립트 화면 우측 상단의 [배포] > [새 배포]를 누릅니다.
  2. 유형 선택(톱니바퀴)에서 [웹 앱]을 선택합니다.
  3. 설명 입력 후, '다음 사용자 권한으로 실행'은 '나(본인 계정)'로 설정합니다.
  4. '액세스할 수 있는 사용자'는 반드시 '모든 사람(Anyone)'으로 지정해 줍니다.
  5. [배포] 버튼을 누르고 권한 검토(승인)를 마칩니다.
  6. 최종 화면에 출력되는 "웹 앱 URL(Web App URL)" 주소를 복사해 둡니다.
4

이곳 웹앱에 주소 입력 및 연동 테스트

복사한 주소를 상단의 '웹앱 주소(Web App URL) 입력' 칸에 넣어 저장하세요. 이후에 제출하는 모든 설문 데이터는 실시간 통계뿐 아니라 선생님의 구글 시트로 직접 들어가 쌓이게 됩니다.
